## 🏗️ 技术架构总结

### 前端技术栈
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI组件库**: Ant Design
- **状态管理**: Redux Toolkit
- **路由**: React Router v6
- **HTTP客户端**: Axios
- **图表库**: Chart.js / ECharts
- **地图服务**: 高德地图API

### 后端技术栈
- **运行时**: Node.js + TypeScript
- **Web框架**: Express.js
- **数据库**: PostgreSQL + Redis
- **ORM**: Sequelize
- **认证**: JWT + Passport.js
- **文件上传**: Multer + Sharp
- **邮件服务**: Nodemailer
- **短信服务**: 阿里云SMS
- **支付集成**: 微信支付 + 支付宝

### 部署技术栈
- **容器化**: Docker + Docker Compose
- **Web服务器**: Nginx
- **进程管理**: PM2
- **监控**: Prometheus + Grafana
- **日志**: Winston
- **安全**: Helmet + CORS
