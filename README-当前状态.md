## 🏃‍♂️ 快速启动

### 1. 启动前端服务
```bash
cd frontend
npm run dev
```
访问: http://localhost:3000

### 2. 启动后端服务
```bash
cd backend
npm install
npm run dev
```
访问: http://localhost:5001

### 3. 查看项目状态
访问: http://localhost:3000/status
